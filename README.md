# Weather-Analysis-Dashboard
Interactive Power BI Weather Analysis Dashboard providing insights into temperature, humidity, wind speed, visibility, UV index, rainfall, and weather forecasts using dynamic visualizations and DAX-driven KPIs.

## Overview
This Power BI project analyzes weather data through an interactive dashboard designed to monitor current conditions and forecast trends. The dashboard provides insights into temperature, humidity, wind speed, visibility, UV index, rainfall, and sunrise information using visually rich reports and KPI cards.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
- Weather Data Analysis

## Features
- Current weather overview
- Temperature trend analysis
- Humidity and rainfall monitoring
- Wind speed and visibility tracking
- UV index insights
- Forecast-based weather analysis
- Interactive filters and visualizations

## Project Structure
- Dashboard (.pbix file)
- Dataset files (.xlsx)
- Icons and background assets
